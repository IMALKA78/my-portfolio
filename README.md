# Imalka Ireshan | Advanced UI/UX & Frontend Portfolio

A high-performance, immersive portfolio website built to showcase UX research and frontend engineering expertise. This site utilizes advanced CSS techniques, Vanilla JavaScript physics, and interactive canvas elements to create a premium user experience.

## ‚ú® Advanced Features

- **Interactive Starfield Background:** A custom-coded `<canvas>` engine that creates a high-performance, animated starfield without impacting browser speed.
- **3D Perspective Cards:** Project cards utilize real-time mouse tracking to create a 3D tilt effect, adding physical depth to the UI.
- **Magnetic Physics Interactions:** Call-to-action buttons feature magnetic attraction logic, "pulling" toward the cursor for increased engagement.
- **Liquid Custom Cursor:** A dual-layer custom cursor system with an elastic aura and `mix-blend-difference` logic for high-contrast visibility across all sections.
- **Responsive Motion Design:** Utilizes the Intersection Observer API for smooth, performance-optimized scroll-reveal animations.
